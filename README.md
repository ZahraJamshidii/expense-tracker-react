# 💰 Expense Tracker Dashboard

A modern Expense Tracker Dashboard built with React.js that helps users manage their personal finances.

---

## ✨ Features

- 🔐 User Authentication (Register & Login)
- 👤 User Profile
- 💵 Add Income & Expenses
- ✏️ Edit Transactions
- 🗑 Delete Transactions
- 🔍 Search Transactions
- 🗂 Filter by Category
- 📅 Filter by Date Range
- 📊 Pie Chart Reports
- 📈 Line Chart Reports
- 📂 Category Management
- 💾 LocalStorage Persistence
- 📱 Responsive Design

---

## 🛠 Technologies

- React.js
- React Router DOM
- React Hook Form
- Context API
- Tailwind CSS
- Recharts
- React Icons
- LocalStorage
- Vite

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/expense-tracker-react.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
